Ubuntu 命令行中文环境配置指南

Posted on 12.07.2022 in os • 84 words • 1 minute read
Tags: ubuntu

1. 检查当前语言环境

echo $LANG

如果显示 en_US.UTF-8，表示当前为英文环境，需要进行中文环境配置。

2. 安装中文语言包

# 更新软件源并安装中文语言包
sudo apt-get update
sudo apt-get install language-pack-zh-hans

3. 配置系统语言环境

3.1 修改locale配置

sudo vim /etc/default/locale

替换文件内容为：

# 系统语言环境设置
LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh"

# 区域设置
LC_NUMERIC="zh_CN"
LC_TIME="zh_CN"
LC_MONETARY="zh_CN"
LC_PAPER="zh_CN"
LC_NAME="zh_CN"
LC_ADDRESS="zh_CN"
LC_TELEPHONE="zh_CN"
LC_MEASUREMENT="zh_CN"
LC_IDENTIFICATION="zh_CN"

# 全局语言环境
LC_ALL="zh_CN.UTF-8"

3.2 修改环境变量配置

sudo vim /etc/environment

在原有PATH设置下方添加：

# 保持原有的PATH设置不变
PATH="..."

# 添加中文环境设置
LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh"
LC_NUMERIC="zh_CN"
LC_TIME="zh_CN"
LC_MONETARY="zh_CN"
LC_PAPER="zh_CN"
LC_NAME="zh_CN"
LC_ADDRESS="zh_CN"
LC_TELEPHONE="zh_CN"
LC_MEASUREMENT="zh_CN"
LC_IDENTIFICATION="zh_CN"
LC_ALL="zh_CN.UTF-8"

4. 重启系统

sudo reboot

注意事项

  1. 配置文件中不要包含多余的空格
  2. 修改配置文件时需要使用sudo权限
  3. 所有修改必须重启系统后才能生效
  4. 如果重启后仍未生效，可以尝试重新执行：
sudo locale-gen zh_CN.UTF-8

验证配置

重启后，可以通过以下命令验证语言环境：

echo $LANG
locale

如显示为中文相关配置，则表示配置成功。
