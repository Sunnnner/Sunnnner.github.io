Python虚拟环境pipenv使用

Posted on 07.07.2022 in python • 20 words • 1 minute read
Tags: pyenv

  • pipenv 是一个包的版本控制软件，与我之前用过的rust语言中的cargo类似，与go的包管理器也类似，它能够锁定当前包的依赖项

  • Ubuntu中可以使用sudo apt install pipenv

  • 也可以使用当前python环境中的pip进行安装pip install pipenv

  • pipenv 命令

  • 指定python版本 pipenv --python 3.8.12

  • 使用当前requements文件生成Pipfile文件pipenv install

  • 锁定当前Pipfile文件pipenv lock

  • 详细信息请参阅https://pipenv.pypa.io/en/latest/
