set

  • set特点：
  • 可以将一个元组转换为set集合并且是每个字符都分开以逗号相隔
  • 2、去重复并且打乱里面的排序
  • 3、x&y（交集）比较集合里的元素有相同的就取出相同的没有相同的返回set()
  • 4、x|y（并集）合并两个集合y在x的后面
  • 5、x-y（差集） 取第一个集合的值如果这两个集合有重复的就去点重复的还是取x的集合的值
  • 6、x^y（对称差）去掉重复的合并两个集合
  • 7、交集有长度

  • 使用set（）进行列表取重复
