Python多态

Celery任务队列

celery 简介 任务队列 1.任务队列是一种在线程或机器间分发的任务的机制 2.消息队列的输入是工作的一个单元，称为任务，独立的职程（Worker）进程持续监视队列中是否有需要处理的新任务。 3.Celery 用消息通信，通常使用中间人（Broker）在客户端和职程间斡旋。这个过程从客户端向队列添加消息开始，之后中间人把消息派送给职程 4.Celery 系统可包含多个职程和中间人，以此获得高可用性和横向扩展能力。
